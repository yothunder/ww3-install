# WAVEWATCH III® Installation

This repository contains a short guide to install WAVEWATCH III® model using BMKG-OFS configuration. 

## Requirements

Libraries
- Compiler
- CMAKE
- NETCDF
- HDF5
- OPENMP & MPI

## Steps

Follow the installation from `compile.sh`, adjust the module, compiler, and path based on your system. This file initialy used to compile the model on a cluster, remove or comment the module part if you are running on your pc or laptop.
